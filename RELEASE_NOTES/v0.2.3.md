Kleines Nachschieben zur v0.2.2: das Plugin nennt jetzt seine
GitHub-Releases-URL als **`UpdateChannelUrl`** in der `PluginMetadata`,
damit der Cockpit-Auto-Update-Check (ab Cockpit v1.7.2) das Plugin
mitprüfen und im Plugin-Updates-Dialog aktualisieren kann.

Voraussetzung: Cockpit ≥ **v1.7.2** und die aktualisierte
`Checkmk.PluginContracts.dll` im `deps/`-Ordner (ist in dieser Release
mitgeliefert).

Wer Cockpit v1.7.2+ nutzt: einmalig diese Version installieren, danach
läuft das Plugin-Update automatisch über den Cockpit-Dialog
„Plugin-Updates…" im Über-Fenster.
